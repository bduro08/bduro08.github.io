Completed Lab

Active Directory Domain Controller Lab

I built a Windows Server 2022 domain controller inside a virtual machine on my Linux Mint machine, created a FakeBank.local domain, and joined my Windows laptop to the domain as an employee workstation.

The lab focused on core identity and access administration: installing Active Directory Domain Services, organizing users into OUs, creating employee and IT administrator accounts, applying Group Policy, enforcing password and lockout rules, updating policy from the command line, and verifying audit events in Event Viewer.

Windows Server 2022 Active Directory Group Policy Domain Join Account Security Audit Logging

Lab Outcome

A working local domain with managed users, a joined Windows 11 endpoint, password and lockout policy enforcement, and security audit evidence.

In Progress

Authorized Phishing-Awareness Simulation

I am building an isolated, local-only lab using a mock company and mock adversary to simulate phishing scenarios against myself. The lab will include a mock email workflow and a mock company website built with Python, Flask, and HTML.

The goal is to observe form interactions, session behavior, and logging in a controlled environment, then turn the results into threat modeling notes, detection ideas, remediation steps, and user-awareness training.

Python Flask HTML Logging Threat Modeling Security Awareness

Ethical Scope

This project uses fictional companies, fictional data, and an isolated local environment. No real user data is collected and no testing targets real organizations.
